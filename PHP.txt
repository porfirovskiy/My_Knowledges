/etc/php/ - в большинстве случаев основной каталог где размещаются разные версии пхп.


php.ini - хранятся основные настройки. В режиме работы модуля веб-сервера, считывается один раз при запуске веб-сервера.
В остальных режимах считывается при каждом вызове.

php-fpm.conf - хранятся глобальные настройки для php-fpm

fpm/pool.d/www.conf - хранятся настройки для пулов дочерних процессов. Их может быть несколько(blog.conf, forum.conf).
Каждый пулл может слушать свой порт и иметь свои настройки.
